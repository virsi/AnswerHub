Как правильно настроить Docker для Django проекта?

Пытаюсь настроить Docker для моего Django приложения, но возникают проблемы с миграциями и статическими файлами. Какие best practices существуют для такой настройки?

В частности интересует:

  • Оптимальная структура Dockerfile
  • Настройка docker-compose для разработки
  • Обработка статических файлов в production
docker django python

Для Django с Docker рекомендую следующую структуру:

FROM python:3.11
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
CMD ["python", "manage.py", "runserver"]

Для миграций добавьте в docker-compose команды для выполнения migrate при запуске.
